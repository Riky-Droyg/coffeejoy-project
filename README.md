# CoffeeJoy — Specialty Coffee Landing Page ☕️

A responsive landing page for a specialty coffee brand. Built as a team project with a focus on clean layout, smooth section navigation, and a polished UI for a modern marketing website.

## Live Demo
https://riky-droyg.github.io/team-project-04/

## Features
- Responsive layout (mobile / tablet / desktop)
- Section-based navigation (About, Benefits, Gallery, Testimonials, Location)
- Gallery block with visuals and neat spacing
- Testimonials carousel/slider
- Newsletter subscription form with validation and user feedback
- Contact & location section with quick access to directions and social links

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Vite
- Git / GitHub
